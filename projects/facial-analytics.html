Overview

FaceMetric is a real-time facial analysis system that performs automated face detection followed by age group and gender classification. The application supports both webcam streaming and static image input through an interactive GUI.

The project demonstrates practical deployment of deep neural networks for visual recognition tasks using OpenCV’s DNN framework and pre-trained Caffe models, integrated into a desktop interface for real-world usability.

Key Features

01

Real-Time Face Detection

Detects multiple faces from live webcam feed or uploaded images using OpenCV’s deep learning face detector.

02

Age Group Prediction

Classifies detected faces into predefined age ranges using a trained Caffe-based neural network.

03

Gender Classification

Predicts gender from facial features using a dedicated deep neural model.

04

Interactive GUI

Provides image upload and webcam modes through a Tkinter-based desktop interface.

Implementation

  • Face detection performed using OpenCV DNN face detection model.
  • Pre-trained Caffe networks used for age and gender classification.
  • Facial regions extracted and passed through DNN pipelines.
  • Tkinter GUI handles user input and visualization.
  • Bounding boxes and prediction labels rendered on output frames.
