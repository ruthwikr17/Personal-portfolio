PostX — AI Tweet Generator

An AI-powered system that generates and posts high-quality tweets automatically

View Code Live Demo

Overview

PostX is an end-to-end AI automation system that generates high-quality tweets from natural language prompts and posts them directly to a demo X (Twitter) account. The project demonstrates real-world integration of modern generative AI APIs with social media automation pipelines.

The goal of PostX was to build a complete AI-powered application from prompt engineering and text generation to API-based publishing and cloud deployment while maintaining an interface visually inspired by X’s dark theme. It serves as both a functional automation tool and a full-stack portfolio project.

Key Features

01

AI-Powered Tweet Generation

Uses Google Gemini’s generative language model with custom prompt engineering to produce concise, high-density tweets optimized for X’s 280-character limit.

02

Editable Tweet Preview

Provides a real-time tweet preview interface where users can edit generated content before publishing.

03

Automated Posting to X

Integrates with X’s API using OAuth authentication to automatically publish generated tweets to a demo account.

Implementation

The system is designed as a lightweight full-stack pipeline connecting an AI text generation service with a social media automation layer and a custom web UI.

  • Prompt-engineered system instructions to control tone, length, and structure of generated tweets.
  • REST API calls to Google Gemini for text generation.
  • OAuth-based authentication and posting via X API using Tweepy.
  • Streamlit-based UI replicating X’s dark theme for user interaction.
  • Cloud deployment on Streamlit for public accessibility.
