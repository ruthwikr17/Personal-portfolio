Semantic Plagiarism Detection System

A semantic analysis tool using NLP and OCR to detect content similarity across documents and images

March 2025 • NLP / Computer Vision
View Code Live Demo

Overview

This project is an advanced plagiarism detection dashboard built with Streamlit. Unlike traditional tools that rely on exact keyword matching, this application leverages Semantic Analysis (using Sentence Transformers) to detect paraphrased content and conceptual similarities between documents.

The system features a versatile file handler capable of processing PDFs, Word documents (.docx), and even images via OCR (Optical Character Recognition). It provides a clean, user-friendly interface for uploading two sources, previewing their content natively in the browser, and receiving an instant similarity score with a detailed verdict.

Key Features

01

Semantic Similarity Analysis

Uses the paraphrase-MiniLM-L6-v2 model to generate vector embeddings, allowing the system to detect plagiarism even when words are synonym-swapped or rephrased.

02

Multi-Format & OCR Support

Supports upload of PDFs, DOCX, TXT, and Images (JPG/PNG). Integrated Tesseract OCR extracts text from image-based documents for analysis.

03

Native Document Preview

Includes a popup modal viewer that renders PDFs and images directly within the app, allowing users to visually verify the uploaded content without downloading it.

Implementation

The pipeline integrates computer vision and natural language processing to normalize data before analysis.

  • Text Extraction: Utilizes PyPDF2 for PDFs, python-docx for Word files, and pytesseract for image OCR.
  • Preprocessing: Cleans text by removing special characters and normalizing whitespace using RegEx.
  • Vectorization: Encodes processed text into high-dimensional vectors using HuggingFace Sentence Transformers.
  • Comparison: Computes Cosine Similarity between vectors to derive a match percentage.
  • UI/UX: Implements Streamlit Dialogs for clean popups and custom iframes for PDF rendering.

Challenges & Learnings

Visualizing Document Previews

Streamlit does not natively display PDF files easily. I solved this by encoding the PDF to Base64 and embedding it within an HTML <iframe> inside a modal dialog.

OCR Deployment on Cloud

Deploying Tesseract on cloud platforms required configuring binary dependencies (packages.txt) alongside Python libraries, a crucial learning step in cloud DevOps.

Key Learnings

This project deepened my knowledge of vector embeddings, cosine similarity math, and the complexities of handling binary file streams in a web application context.
